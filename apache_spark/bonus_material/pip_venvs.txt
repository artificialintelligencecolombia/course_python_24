==============================
VENVS IN PIP
==============================

---------------------
Basic Management
---------------------

1. Ensure python is installed
    python --versiob

2. Create a venv 
    python -m venv myenv

3.  Activate it 
myenv\Scripts\activate 

4. Install packges using pip 
    pip install <package_name>

5. Deactivate venv 
    deactivate
